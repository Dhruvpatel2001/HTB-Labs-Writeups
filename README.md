# HTB-Labs-Writeups
## 🔐 About This Repository

This repository documents my hands-on journey into cybersecurity through Hack The Box labs.  
Each write-up reflects a structured approach to penetration testing, including reconnaissance, enumeration, exploitation, and post-exploitation.

Rather than just solving machines, I focus on understanding:
- Why a vulnerability exists  
- How it can be exploited  
- What security principles are being broken  

The goal is to build practical skills in ethical hacking and develop a strong foundation for real-world penetration testing.
